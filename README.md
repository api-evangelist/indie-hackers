# Indie Hackers (indie-hackers)

Indie Hackers (https://www.indiehackers.com/) is a community for founders building profitable online businesses. Founded by Courtland Allen in 2016 and acquired by Stripe in 2017, the platform provides a space where entrepreneurs openly share revenue numbers, business strategies, and lessons learned from building independent, often bootstrapped products. The community centers on transparency — members regularly post MRR and ARR figures, discuss challenges, and support each other through forums, groups, interviews, and a widely followed podcast and newsletter.

## APIs

### Indie Hackers API

The Indie Hackers API provides programmatic access to the Indie Hackers community platform, enabling developers to connect with a network of founders who openly share the strategies and revenue numbers behind their companies and side projects. Authentication is handled via API key, and the API exposes community data including posts, products, interviews, and founder profiles for builders working toward profitable, independent online businesses.
